Introduction

The Ackermann Controller is a C++17 library that implements steering control for vehicles using Ackermann steering geometry. It converts high-level planner commands (linear velocity v and angular velocity Ï‰) into actuator-level steering and wheel velocity outputs for autonomous vehicles.

Key Features

  • Ackermann steering kinematics computation
  • Configurable vehicle parameters:
    • Drive width and length
    • Wheel radius
    • Steering angle limits
    • Velocity limits
  • PID control for wheel speed control
  • Comprehensive unit testing suite
  • Test-Driven Development (TDD) approach

Architecture

The project is organized into two main libraries:

  • libackermann: Core implementation of Ackermann steering controller
  • libpid: Implementation of PID controller for wheel speed control

Usage

Here's a basic example of using the Ackermann Controller:

#include "ackermann_controller.hpp"
// Create controller instance with vehicle parameters
AckermannController controller(wheelbase, track_width, wheel_radius);
// Set desired velocities
double linear_velocity = 1.0; // m/s
double angular_velocity = 0.5; // rad/s
// Get wheel velocities and steering angles
controller.computeWheelSpeedsAndSteering(linear_velocity, angular_velocity);
