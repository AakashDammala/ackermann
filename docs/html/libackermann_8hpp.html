Detailed Description

PID-integrated Ackermann controller and kinematic model.

This module provides: A lightweight kinematic model (AckermannModel) for an Ackermann vehicle that evolves the state and computes per-wheel outputs (RPM and steering angles). A controller (AckermannController) that converts vehicle-level references — linear velocity v [m/s] and yaw rate ω [rad/s] — into front-wheel steering angles and per-wheel RPM using two scalar PID controllers (speed and steering).

Conventions and assumptions:

  • Positive angular velocity (ω > 0) implies a left/CCW turn.
  • Front-axle bicycle model is used for steering geometry.
  • Outputs are clamped to configured velocity and steering limits and may be rate-limited and dead-banded.
