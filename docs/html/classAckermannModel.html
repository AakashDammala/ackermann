Detailed Description

Simple bicycle-model kinematics with per-wheel mapping.

Responsibilities:

  • Update the vehicle state with a fixed time step and steering input.
  • Map the current motion/geometry to per-wheel linear speeds and steering angles, then to RPM via wheel radius.

The heading is typically kept wrapped using AckUtils::WrapTo2Pi.
