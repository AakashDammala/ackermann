Detailed Description

Two-loop controller (speed + steering) that feeds Ackermann kinematics.

Pipeline (per call to Compute):

  1. Speed loop: PID on longitudinal speed using previously applied v as the “measurement”; apply clamp → deadband → rate limit.
  2. Steering reference: δRef = atan2( L·ω , max(|v|, ε) ).
  3. Steering loop: PID on steering angle using previously applied δ as the “measurement”; apply clamp → deadband → rate limit.
  4. Kinematics: Map (v, δ) to per-wheel RPM and front steering angles.

PID implementations are injected (dependency injection) to keep the controller testable and to decouple PID details from kinematic/geometry logic.
