void PIDController::SetAntiWindupGain ( double  Kaw)

Configure anti-windup back-calculation gain.

Parameters
KawAnti-windup gain ( -like scaling). Set 0 to disable.

When enabled, the integrator receives in addition to the nominal integral update, reducing windup when outputs saturate.
